# 香港 BIM 數碼交付 + ACC 檢查清單

**用途**：開新項目時及交付前，對照香港指引與 ACC 操作，確保資料夾、命名、表單、權限與交付包符合要求。  
**對照詳情**：見 [《香港 BIM / 數碼交付指引與 ACC 對照表》](HK_BIM_Digital_Delivery_ACC_Mapping.md)。

---

## 一、開新項目時（項目啟動）

| # | 香港要點 | ACC 操作 | 勾選 |
|---|----------|----------|------|
| 1 | 使用符合香港的項目模板 | 建立項目時選「香港住宅」或「香港基礎設施」模板 | ☐ |
| 2 | 項目類型與合約類型一致 | Project type 選 Residential/Buildings 或 Infrastructure（如 Roads）；工務項目確認 TC 1/2025 適用 | ☐ |
| 3 | 資料夾階段劃分（DEVB 附錄 III） | 確認模板已含 WIP / Shared / Published（或等同）及招標/合約、參考與標準、EIR-BEP、提交 LandsD（若適用） | ☐ |
| 4 | 命名規則對應 DEVB / BD | 確認 File Naming Standards 已套用；說明 BD 格式與軟件版本（ADM-19/ADV-34） | ☐ |
| 5 | 權限對應 IRAM（DEVB 附錄 II） | 檢查 Folder Permissions 已按角色/公司設定，與資訊責任一致 | ☐ |
| 6 | EIR / BEP / PIR 存放位置 | 在「參考與標準」或專用資料夾放置項目 EIR、BEP、PIR（或連結） | ☐ |
| 7 | 表單就緒（EIR 檢查、BEP、收尾、提交） | 確認 Forms 已從 Library 或模板帶入：EIR 條款檢查、BEP 里程碑、項目收尾清單、LandsD/BD 提交檢查表（按項目類型） | ☐ |

---

## 二、交付前（標書 / 設計 / 竣工 / 提交政府）

| # | 香港要點 | ACC 操作 | 勾選 |
|---|----------|----------|------|
| 8 | 標書含 BIM、合約化 BIM（TC 1/2025） | 「招標與合約」資料夾內 BIM 與圖則齊備；使用「標書 BIM 交付清單」或「合約化 BIM 版本確認」Form 簽認 | ☐ |
| 9 | 設計/竣工 BIM 提交 LandsD（工務） | 「提交 LandsD」資料夾內檔案符合附錄 XIV；完成 LandsD 提交檢查清單 | ☐ |
| 10 | 屋宇署 BD 呈交（私營） | 法定圖則與 BIM 補充分開標示；完成 BD 提交檢查表；命名與格式符合 ADM-19/ADV-34 | ☐ |
| 11 | 資訊容器命名與狀態 | 檔案命名符合 DEVB 附錄 VIII–X；版本/修訂在 Docs 可追溯；需狀態碼時在流程或 Authoring 標註 | ☐ |
| 12 | 項目收尾（DEVB 附錄 XII） | 完成 Project Close-out Checklist；必要文件已歸檔至對應資料夾 | ☐ |
| 13 | BIM 物件檢查（DEVB 附錄 VII，若適用） | 完成 BIM Object Check Form 或等同檢查表 | ☐ |

---

## 三、備註

- **工務項目**：上表 1–9、11–13 直接適用；須符合 DEVB 統一指引、TC 1/2025、附錄 XIV（LandsD）。
- **私營項目**：1–7、10–12 為主；CIC、BD 為參考；8–9、13 按合約或業主要求。
- **在 Authoring 完成**：LOD、屬性、BD 自動核查、結構工具、檔案名稱試算表驗證等在 Revit/Civil 3D/ArchiCAD 或外部系統完成；ACC 負責存放、版本、權限與清單勾選。

*本清單為實務輔助，不取代合約或法定要求；以最新官方文件為準。*
