# 6 Standardising View Setting

98  
6 Standardising View Setting 
This template has defined the desired settings. The users can use view 
templates to manage these settings  by applying the view template, user 
efficiency on projects enhanced by assigning default view template for each view 
type. The standardised view propert ies include, but not limit ed to, view scale, 
detail level, model display setting. 
 
Open either view plan ➜ On the “View” tab ➜ find the “Graphics” panel ➜ select 
“View Templates” ➜select “Apply Template Properties to Current View” 
 
 
In “View Templates” dialogue box ➜ on the right, select the view template 
applied to the corresponding view ➜ click “OK” to close the window. 
Changes to view templates are automatically reflected in the views to which they 
have been assigned.
