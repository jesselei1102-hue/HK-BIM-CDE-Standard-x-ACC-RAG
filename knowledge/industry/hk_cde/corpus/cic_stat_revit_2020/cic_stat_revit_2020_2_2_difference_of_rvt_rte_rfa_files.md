2.2 Difference of RVT, RTE, RFA files 
It will eventually become necessary to save our work and share our progress 
with others in the model. RTEs are project template file. RVT s are working 
project files. Both files can be generated from Revit® projects. The difference 
is that the template is used when you start a new project.  
RFAs are family files that can either be loaded into a project or saved 
externally.
