LA.4 Information Container Metadata 
 
LA.4.1 Clarification 
ISO 19650‑2:2018 (5.1.7.c) states: 'The project's common data environment shall enable each 
information container to have the following attributes [metadata] assigned: status (suitability); 
revision; classification (in accordance with the framework defined in ISO 12006-2)'. 
In the Hong Kong, attributes (metadata) for information containers within a common data 
environment should be defined from the following codification table.
