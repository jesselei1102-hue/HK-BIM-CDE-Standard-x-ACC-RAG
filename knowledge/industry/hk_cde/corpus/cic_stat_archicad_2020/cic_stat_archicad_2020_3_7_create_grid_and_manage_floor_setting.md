# 3.7 Create Grid and Manage Floor Setting

23 
 
3.7 Create Grid and Manage Floor Setting  
 
Based on the drawing, add the grid symbol in the toolbox. In the default dialogue box, 
the user can select the select detail setting based on their needs for both floor plan 
and section plan.

24 
 
 
 
For floor setting or story setting in ArchiCAD, Select DesignStory Setting. In the 
setting, box defines the floor height and elevation of each floor in your project.
