# 3.2 Set MEP Working Environment

14 
 
 3.2 Set MEP Working Environment 
 Click the File Library and Object Library Manager 
 
 Click Add to import the MEP Library 23 or any customized library. And click OK.

15 
 
In order to show the MEP tool in the toolbox, Select OptionWorking Environment  
Apply Profile MEP Profile.

16 
 
 To quick access the MEP edited tool, Select WindowToolbarMEP Toolbar 
 
 
               Click the MEP System Setting .

17 
 
In MEP settings, create the drainage systems and choose it to use with pipework.   
The Edit Select System is mainly used to distinguish the system by giving a different 
colour. For drainage plan, we did it as shown. Later we mainly use the graphical 
override function to display only the Axis lines.  
 
Click the Graphical Override on the bottom toolbar. 
 
In the dropdown menu, Choose Drainage Plan only to display the Axis Line. 
Choose No Overrides in order to display the Elements Fill on 2D.
