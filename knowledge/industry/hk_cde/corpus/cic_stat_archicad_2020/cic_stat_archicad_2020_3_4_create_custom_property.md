# 3.4 Create custom property.

19 
 
 3.4 Create custom property.  
The template is the reach of detail properties related to drainage plan. If you want to 
add extra properties, select OptionProperty Manager 
 
 
 
In Property Manager, you can create a new property by click the new bottom on the 
left.  
 
The user can also edit in detail of the created property. Like to define this property 
will available for which classification item. 
 
• For Classification function, please refer to 2.9 Property and Classification
