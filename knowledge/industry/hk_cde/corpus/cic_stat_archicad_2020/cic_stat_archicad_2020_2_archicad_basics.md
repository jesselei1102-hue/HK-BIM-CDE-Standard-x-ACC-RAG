# 2. ArchiCAD Basics

2 
2. Ar chiCAD Basic s
2.1 User Interface
• In ArchiCAD, for all the projects, the user interface provides all the tools to work
with the model. The user interface can be customized to feed various kinds of
works.

3 
 
• Pop-Up Navigator: manage the floor plan, view, schedules, layout plan and 
publish setting of the project. 
 
 
• Bottom Toolbar: help to navigate the view, set the scale of the drawing and zoom 
factor.

4 
 
• Menu Bar & Standard Toolbar: all the commands and functions are listed for BIM 
modelling.  
 
• Toolbox: List all the tool include MEP for building the 3D model and prepare for 
the documentation 
 
 
• Element Default Setting: manage the properties (include IFC) of the selected 
element.
