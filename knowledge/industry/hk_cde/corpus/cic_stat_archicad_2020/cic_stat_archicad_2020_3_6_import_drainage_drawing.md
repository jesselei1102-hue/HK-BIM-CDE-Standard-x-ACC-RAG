# 3.6 Import Drainage Drawing

21 
 
The ArchiCAD mainly support three types of location data: 
The XML format 
 
The Google Earth (.kmz) format 
 
The SketchUp (.skp) format 
 
 
 
 
3.6 Import Drainage Drawing  
Create a new worksheet in the Project Map and give a name like DR-1F. Click the 
worksheet in the Project Map

22 
 
Click “File” “External Content””Place External Drawing”.  
 
 
 
Choose the drawing in the folder and place the drawing. 
 
Select the units of the drawing according to the project requirement.
