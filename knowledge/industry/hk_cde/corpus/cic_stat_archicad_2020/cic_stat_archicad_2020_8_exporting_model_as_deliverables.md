# 8. Exporting Model as Deliverables

47 
 
8. Exporting Model as Deliverables  
 8.1 Exporting Layout as PDF and DWG 
In Organizer, Choose one side as Layout Book one side as Publisher Sets.  Create a 
new publish set to publish the layout.  
 
 
 
Drag the layout you want to publish into the Publish Set you create.  
 
Double click the set and choose the publish format as PDF or DWG and do some other 
customization for your PDF or DWG.  
 
Click Publish button to create the PDF or DWG.
