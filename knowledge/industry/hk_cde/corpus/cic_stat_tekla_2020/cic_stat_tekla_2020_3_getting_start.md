3 Getting Start 
Users are expected to install Tekla 2019i before followings the steps stated in this section. 
3.1 Opening Project Template 
Download template file from CIC website. Run the Env_SE_Asia_2019.exe to install Asian Tekla 
tool set. Put the folder set ““CIC_TEMPLATE_STR_2019”” to C:\ProgramData\Trimble\Tekla 
Structures\2019.0\Environments\SE_Asia\Precast\model_templates. 
 
Open Tekla 2019, open new file with template “CIC_DEMO_MODEL_TEMPLATE” being chosen. 
 
 
3.2 Set-up of Project Base Points
