4.14 Attaching 3D topographic models   
 
 Tekla is not designed to model 3D topographic models,  but users could attach 3D topographic 
models from other software to Tekla through the ‘Reference Model: function. Users need to name 
those reference model systematically in order to manage their visibility in 2D drawings.
