# 4.7 Creating Piles

14 
 
4.7 Creating Piles 
Concrete piles are created in Tekla by using concrete columns while steel piles are created by 
using steel columns, please refer to section 4.2 in this user guide for detail.  
Users may download the Bellout.uel profile file from the CIC website if they want to model bored 
piles with bellout. Import the uel file from “Profile Catalog” which could be found in the file menu at 
the top right corner (see the user interface at “General” in this user guide). “Profile Catalog” is 
under the catalogues tag.
