# 4.13 Creating Monitoring Instruments

21 
 
4.13 Creating Monitoring Instruments  
It is suggested to create a simple box model in Tekla to represent geotechnical monitoring 
instruments such that objects could be scheduled if needed. Users could choose whether to insert 
3D instrument model by using “shape” in “concrete items” as mentioned in section 4.12 in this user 
guide or simply use native concrete parts to create a box model such that the space consumed by 
the instrument could be adjusted i nside Tekla. Name the instrument in the name entry in property 
menu such that the name could be shown by using associative note in the drawing. 
 
It is suggested to use a symbol in part mark to denote the location and type of the instrument and 
use an associative note to denote the name. Such that information could be shown clearly on 2D 
drawings, even some instruments are put very close to each other.
