# Tie beams should be modelled by using strip footing under the Concrete > Footing tab.

15 
 
4.8 Creating Pile Caps and Tie Beams  
Pad footing is used to create a pile cap in Tekla. It is suggested to create a one big rectangular 
box model by using pad footing and then trim it by using polygon cut. It is not suggested to use 
more than one pad footing to model one pile cap as this would cause problems during 2D drawi ng 
production.   
 
 To minimise the use of polygon cuts, it is suggested to use custom profiles to create footing with a 
shape which can be created by simple extrusions, such as the triangular pile cap as shown below. 
Users may use “define cross-section in sketch editor” to create custom profiles.  
 Tie beams should be modelled by using strip footing under the Concrete > Footing tab.
