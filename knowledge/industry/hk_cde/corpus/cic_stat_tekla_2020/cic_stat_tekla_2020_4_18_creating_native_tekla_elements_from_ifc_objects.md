# 4.18 Creating native Tekla elements from IFC objects

4.17 Creating Curtain Wall Elements 
Curtain wall elements such as transoms and mullions can be either modelled by simple rectangular 
sections (e.g. RHS) or the actual profiles can be imported to Tekla. For the latter, users can create 
user-defined sections by sketching the section or importing from a DWG file. 
 
4.18 Creating native Tekla elements from IFC objects 
It is possible to convert IFC reference model objects to native Tekla elements using the “Convert 
IFC objects” tool. Users can choose to convert a portion of , or the entire IFC mode l. Users can 
overview the converted properties and choose the conversion type.
