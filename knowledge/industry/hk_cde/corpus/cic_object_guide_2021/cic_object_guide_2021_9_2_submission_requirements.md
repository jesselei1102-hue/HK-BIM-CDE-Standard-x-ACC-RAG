# 9.2 Submission Requirements

CIC PRODUCTION OF BIM OBJECT GUIDE 
24 
9.2 Submission Requirements 
The BIM object shall be provided with a completed BIM object sheet and check form to prove that the object is complete and 
satisfies the requirements of this Guide. 
The BIM object and relevant documents shall be submitted as a package and follow a standardised folder structure, as follows: 
 
Folder Name (The BIM object name without file format extension) 
 ├ The BIM object file 
 ├ Image Folder (Storage of the BIM object geometry and property images) 
 ├ Symbol Folder (Storage of the symbol object file(s) and its image(s))  If Applicable 
 ├ Tag Folder (Storage of the tag object file(s) and its image(s))   If Applicable 
 └ Document Folder (Storage of the completed BIM Object sheet and BIM Object check form)
