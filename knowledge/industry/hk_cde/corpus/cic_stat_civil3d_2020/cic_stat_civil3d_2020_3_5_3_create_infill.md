3.5.3 Create Infill 
When creating Grading, only the part along the selected feature line is created. The inside 
area of the feature line is still not filled. The Infill will help to create this area. 
On Grading Creation Tools toolbar, click on Create Infill 
 
Click on the inner side of the feature line to create Infill for that area. The surface will now
