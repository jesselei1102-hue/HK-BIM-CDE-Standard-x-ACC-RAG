31 
 
2.2.1. Advice on selection of CDE in public or private cloud environment 
 
(B) Advice on selection of CDE in public or private cloud environment 
No. Item Answer Advice on CDE  
1. Does the project involve any sensitive 
project information that is not allowed to be 
stored in public cloud? 
Yes Private cloud  
No Public cloud  
Remark: Companies should conduct any  required internal or external Information Technology (IT) security 
risk assessment to ensure any sensitive project information can be stored in public cloud. 
Remark: For government projects, it is suggested that the CDE should be configured and managed to comply 
with the latest relevant Office of the Government Chief Information Officer (OGCIO)’s Information Technology 
(IT) security requirements , such as Practice Guide for Cloud Computing Security (ISPG -SM04) and the 
prevailing Security Regulations of the Security Bureau. 11  For details, please refer to 
https://www.ogcio.gov.hk/en/our_work/information_cyber_security/government/ 
2. Does the project involve any sensiti ve 
project information that data sovereignty is a 
concern? 
Yes Private cloud  or public cloud  
(that allows buyer to choose 
cloud service provider and 
location of the data center for 
the data storage and hosting of 
the CDE)  
 
No Public cloud  
Remark: Data sovereignty (sometimes known as data residency) refers to the location(s) where the data is 
to be physically stored.  Security concern on data sovereignty can be associated with geopolitical risk that 
may have influence on the governance and operation of the data centre and the cloud service provider, which 
may cause undesirable service delay, disruption or data leakage.  For more information, please refer to  
Practice Guide for Cloud Computing Security [ISPG-SM04], Version 1.2, OGCIO. 
Remark: Some CDE products do not allow customer to have choice of the public cloud service providers . 
3. Is there any overseas consultant or 
production team stationed outside HK that 
needs to access the CDE frequently? 
Yes CDE with appropriate Cloud 
Data Center location. 
 
No Any CDE solution is fine  
Remark: “Appropriate” here refers to the data center location for the storage and hosting of CDE. Some 
countries may have limitations on network accessibility and speed outside their regions (e.g. BIM production 
teams from China that need to access the CDE hosted in Singapore data center).
